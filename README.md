# 🛍️ Myntra Clone

## 📖 About

This project is a **Myntra Clone** built using HTML, CSS, and JavaScript. It replicates the core layout and basic functionality of an e-commerce fashion website, allowing users to browse products, view details, and interact with a shopping interface.

The goal of this project is to strengthen front-end development skills by implementing real-world features like dynamic product rendering, navigation between pages, and basic cart functionality.

---

## 🚀 Features

* 🛒 Product listing page
* 🔍 Clickable product cards with navigation to product page
* 📄 Product detail page
* 🧺 Add to Bag functionality (basic)
* 📱 Responsive layout
* 🎯 Clean UI similar to Myntra

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
